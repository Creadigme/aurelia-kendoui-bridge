# aurelia-kendoui-bridge

A set of [Aurelia](http://aurelia.io) wrappers for [Kendo UI](http://www.telerik.com/kendo-ui)

**NOTE: This plugin is in active development and we would like to hear your feedback**

## Running tests
1. `npm install`
2. `jspm install`
3. `karma start`
